## 1:00 PM â€“ CEO Calls to High-Value Debtors

### Objective

### Standard Operating Procedure

- [ ] **Step 1:** Pull today's **CEO Call Queue** from the dashboard
- [ ] **Step 2:** Review debtor profiles for each call:
  - [ ] Judgment amount and age
  - [ ] Previous contact attempts
