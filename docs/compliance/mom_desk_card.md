```
1. "Of course. We'll communicate by mail only."

3. Flag account: WRITTEN ONLY
4. No more phone calls unless they initiate
5. Confirm by sending first written communication
```

```

```

```
2. Log attorney information
3. "We'll direct future communications to them."
4. Flag account with attorney info
5. Contact attorney, not debtor, going forward
```

### When They Say: "I FILED BANKRUPTCY"

```

4. Hang up professionally

6. Report threats to supervisor immediately
```
